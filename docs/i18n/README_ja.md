## 重要なお知らせ

```
+------------------------------------------------------------------------+
|  これはオリジナルの独立したプロジェクトです                            |
|  * すべてのコードはオリジナルの実装です                                |
|  * すべてのクリーチャーデザインはオリジナルです                        |
|  * パブリックドメインのアルゴリズムを使用：xorshift128+, djb2          |
+------------------------------------------------------------------------+
```

---

## 機能

- **16種類のオリジナルクリーチャー** - スライムリング、フラフォックス、スパークフィン、モスベア、クラウドパップ、クリスタルウィング、エンバークロー、フロストウィスク、ソーンバック、グロウモス、サンドスケール、ストームフェザー、デュードロップ、アイアンポー、ヴァインハート、ナイトシェード
- **6段階のティアシステム** - ブロンズ、シルバー、ゴールド、プラチナ、ダイヤモンド、ミシック
- **スパークルバリアント** - ティアベースのスパークル確率（0.5% - 5%）
- **6つの属性** - バイタリティ、アジリティ、スピリット、ラック、チャーム、フォーカス

---

## インストール

```bash
git clone https://github.com/haotool/pixel-pets-cli.git
cd pixel-pets-cli
npm install
npm run dev -- help
```

---

## 召喚システム

| ティア | 確率 | 基本属性 | スパークル確率 |
|--------|------|----------|----------------|
| [B] ブロンズ | 45% | 10 | 0.5% |
| [S] シルバー | 30% | 20 | 0.8% |
| [G] ゴールド | 15% | 35 | 1.2% |
| [P] プラチナ | 7% | 50 | 2.0% |
| [D] ダイヤモンド | 2.5% | 65 | 3.5% |
| [M] ミシック | 0.5% | 80 | 5.0% |
