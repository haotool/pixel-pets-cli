## 功能特色

- **16 种原创生物** - 史莱灵、绒狐、电鳍、苔熊、云犬、晶翼、烬爪、霜须、棘背、萤蛾、沙鳞、雷羽、露滴、铁爪、藤心、夜影
- **6 级阶层系统** - 青铜、白银、黄金、白金、钻石、神话
- **闪耀变体** - 阶层相关的闪耀概率（0.5% - 5%）
- **6 大属性** - 活力、敏捷、精神、幸运、魅力、专注

---

## 安装

```bash
git clone https://github.com/haotool/pixel-pets-cli.git
cd pixel-pets-cli
npm install
npm run dev -- help
```

---

## 召唤系统

| 阶层 | 概率 | 基础属性 | 闪耀概率 |
|------|------|----------|----------|
| [B] 青铜 | 45% | 10 | 0.5% |
| [S] 白银 | 30% | 20 | 0.8% |
| [G] 黄金 | 15% | 35 | 1.2% |
| [P] 白金 | 7% | 50 | 2.0% |
| [D] 钻石 | 2.5% | 65 | 3.5% |
| [M] 神话 | 0.5% | 80 | 5.0% |
