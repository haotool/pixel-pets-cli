## 功能特色

- **16 種原創生物** - 史萊靈、絨狐、電鰭、苔熊、雲犬、晶翼、燼爪、霜鬚、棘背、螢蛾、沙鱗、雷羽、露滴、鐵爪、藤心、夜影
- **6 級階層系統** - 青銅、白銀、黃金、白金、鑽石、神話
- **閃耀變體** - 階層相關的閃耀機率（0.5% - 5%）
- **6 大屬性** - 活力、敏捷、精神、幸運、魅力、專注
- **配件系統** - 9 種原創配件類型
- **確定性召喚** - 使用種子可重現相同結果
- **ASCII 動畫** - 原創 2 幀動畫

## 安裝

```bash
git clone https://github.com/haotool/pixel-pets-cli.git
cd pixel-pets-cli
npm install
npm run dev -- help
```

## 原創實現聲明

本專案是**原創作品**：

1. **所有 16 種生物** - 原創名稱和設計
2. **所有 ASCII 精靈圖** - 專為本專案創作
3. **階層系統** - 原創 6 階結構與獨特機率
4. **屬性系統** - 原創 6 屬性與鐘形曲線分布

**使用的公有領域演算法：**
- xorshift128+ PRNG - 維基百科有記載
- djb2 雜湊 - Daniel J. Bernstein 創建，公有領域
