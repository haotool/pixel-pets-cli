## Public Domain Algorithms

This project uses the following **public domain algorithms**:

### xorshift128+ PRNG
- A well-documented pseudo-random number generator
- Reference: https://en.wikipedia.org/wiki/Xorshift
- Used in many open source projects including V8 JavaScript engine

### djb2 Hash Function
- Created by Daniel J. Bernstein
- Public domain, widely used
- Reference: http://www.cse.yorku.ca/~oz/hash.html

These algorithms are standard, publicly documented, and freely available for use.
